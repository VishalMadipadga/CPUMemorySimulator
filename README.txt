HOW TO RUN PROGRAM:
1. run command: make
2. run command: ./project1 <FILENAME> <NUMBER OF INSTRUCTIONS>

FILES:
1. Makefile: Contains instructions to create executable program
2. Project1.cc: Contains code that forks the program. Also contains code that parses the input file and acts as Main Memory (reading and writing from to memory)
3. Project1.h: Containts CPU class decleration and declares the two pipes pfds and ps that will be used as well as various header files
4. CPU.cc: Contains the CPU class functions, these contain the operations of the CPU in the program
5. ProjectSummary.txt: Contains project summary
6. sample5.txt: contains instructions to multiply two numbers and show the operation to the screen
